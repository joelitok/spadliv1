Fixed Navigation Offset

A fixed position navigation is taken out of the flow of a page and may overlap the content. To conteract this we've created the "header" ID to be used on a container that falls immediately after the navbar. When this element is included, it adds a extra space to allow the content to begin after the navigation. You can also add content to the header element. This is a great place for a full width background and a page title or other masthead design elements.

For example:

<section id="header"> optional masthead content </section>

Navigation Wrapper

When using the static and fixed navigation style, it's important for the wrapper element to not include the container class. Having the wrapper element is necessary for a main navigation bar style, like the example on this page. Including the container class can have an effect on the padding and spacing that produces unwanted results. It is still allowed to have a container element inside the wrapper, but not on the wrapper.
