Cover Template with Background

Cover is a beautiful full-page template with several custom options. To get started, add the class "cover" to the HTML element in the template file. Next, paste the cover HTML elements into the page and start creating content.

See the code

Sticky footer relative to the container using footer-sticky or fixed to browser using footer-sticky-fixed.

Cover Element HTML

<!DOCTYPE html>
<html lang="en" class="cover">
  <head>
    <title>Cover Template</title>
    <!-- don't forget your style sheets! -->
  </head>
  <body>

    <!-- Start of Cover Element -->
    <div class="cover-wrapper">
      <div class="cover-container">
        <div class="cover-inner">

          <h1>Your Content Here!</h1>

        </div>
      </div>
    </div> <!-- /.cover-wrapper -->

  </body>
</html>

Background Image

Specify a backgrond image with CSS.

.cover-wrapper { background-image: url(my-image.jpg); }

Color Overlay

Add the overlay class to darken the background image.

<div class="cover-container overlay">

Change the overlay color with CSS:

.cover-container.overlay { background-color: rgba(0,131,202,.6); }
