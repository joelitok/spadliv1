Floating Navigation

Adding the class <body class="float-navbar"> produces a navigation overlaying the content at the top of the page.

Combining this with an extra <div class="container"> wrapper and a navigation class of <div class="navbar navbar-static-top"> stops the navigation from stretching full width and makes a cool looking, free floating navigation bar.

<body class="float-navbar">

  <div class="navbar-wrapper">
    <div class="container">
      <div class="navbar navbar-static-top" role="navigation">
        <div class="container">
          <div class="navbar-header">
            <a href="./../index.html" class="navbar-brand">Template</a>
            <button class="navbar-toggle" type="button" data-toggle="collapse" data-target="#navbar-main">
              <span class="icon-bar"></span>
              <span class="icon-bar"></span>
              <span class="icon-bar"></span>
            </button>
          </div>
          <div class="navbar-collapse collapse" id="navbar-main">
            <ul class="nav navbar-nav">
              <li>
                <a href="#">Sample</a>
              </li>
              <li>
                <a href="#">Example</a>
              </li>
            </ul>
          </div>
        </div>
      </div>
    </div>
  </div>
